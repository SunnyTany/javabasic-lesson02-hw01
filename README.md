# Вивід в консоль формування замовлень

Магазин здійснює онлайн торгівлю.

В класі **Main** реалізується функціонал формування замовлень за допомогою метода ***printOrder***.

Виведення в консоль має такий вигляд:
* Order No 1 Client: Alice.
* Product: smartphone,
* price EUR 305.99.
* Address: Moon Street, 10.
* Order No 2 Client: Tom.
* Product: laptop,
* price EUR 570.95.
* Address: Terra Street, 17.
